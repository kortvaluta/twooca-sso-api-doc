# TwooCa SSO API Document

## Prerequisites

- Node.js (see [.node-version](.node-version))

## Development

### Setup

```
$ npm install
$ npm run prepare
```

### Serve

```
$ npm run serve
```

http://localhost:8080

### Build

Build automatically on pre-commit. If you would like to build manually, please follow below.

```
$ npm run build
```

output: [dist/index.html](dist/index.html)
